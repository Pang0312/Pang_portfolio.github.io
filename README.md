# Pang Cheng Siang Portfolio

Professional portfolio website for GitHub Pages.

## How to publish
1. Create a new public repository on GitHub.
2. Upload all files from this folder.
3. Go to Settings > Pages.
4. Choose Deploy from a branch.
5. Select main branch and /root folder.
6. Wait 1-2 minutes for your portfolio link.

## Edit details
- Edit text in `index.html`
- Edit colours and mobile layout in `style.css`
- Edit small interaction effects in `script.js`
- Replace `assets/profile.jpg` if you want a new photo
- Replace `assets/Pang_Cheng_Siang_CV.pdf` if you update your CV
